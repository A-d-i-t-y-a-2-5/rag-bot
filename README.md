# RAG Bot

A minimal starter for a Retrieval-Augmented Generation (RAG) bot that answers questions from documents through a conversational interface.

## Requirements

- Python 3.11 or newer
- Optional: a workflow orchestrator like `uv`, or a virtual environment tool (`venv`, `pipenv`, `poetry`)

## Installation
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
uv sync
```
